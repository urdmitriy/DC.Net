Программа Файл-менеджер
Доступные функции программы:
1) help - помощь;
2) cd (путь до директории) (..)- смета активной директории. Команда принимает в качестве аргумента абсолютный, либо относительный путь.
Доступны ключи: pd 1 (page directory) (по-умолчанию страница 1) смена страницы отображения директорий;
                pf 1 (page file) (по-умолчанию страница 1) смена страницы отображения файлов;
                .. - переход в родительский каталог.
так же для текущей директории доступна команда "setpage pd 2 pf 3" с аналогичными параметрами;
3) mkdir (название каталога)  - создание каталога в текущей каталоге, либо по абсолютному пути;
4) rmdir (название каталога)  - удаление каталога;
5) cp (имя файла) (путь копирования) - копирование файла;
6) rn (имя файла) (новое имя файла)  - переименование файла;
7) cf (имя файла) ("содержание файла") - создание файла с опциональным наполнением;
8) cat (имя файла) - просмотр файла. Вывод содержания небольших текстовых файлов на экран.
